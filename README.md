# C++ Maze Game

An object-oriented maze game implemented in **C++**, developed as a university project.

## Overview
This project implements a simple maze-based game using object-oriented programming principles.
The player controls a hero character navigating through different maze maps.

## Features
- Object-oriented design (Game, GameObject, Hero, Maze)
- Multiple maze maps
- Clear separation of game logic and entities

## Technologies
- C++
- Object-Oriented Programming (OOP)

## How to Build & Run
Compile all source files and run the executable:

```bash
g++ *.cpp -o maze_game
./maze_game
